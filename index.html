GenialnTools

Библиотека для ускорения и замедления выполнения кода

Замедление кода

Используйте декоратор @slowed для контролируемого замедления выполнения функций. Идеально для отладки и анализа.

Ускорение кода

Применяйте декоратор @speed для увеличения скорости выполнения функций. Максимизируйте производительность.

Простота использования

Интуитивно понятный API позволяет легко интегрировать функциональность в существующие проекты.

Примеры использования

Замедление выполнения

from genialntools import Slowed

slowed = Slowed()

@slowed(2)
def test():
    print("Hello")

test()

Ускорение выполнения

from genialntools import Speed

speed = Speed()

@speed(2)
def calc(x):
    return x * 2

print(calc(5))

Установка

Установите библиотеку с помощью pip:

pip install genialntools

Начните использовать возможности ускорения и замедления кода уже сегодня!
